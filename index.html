Política de Privacidade – Géti

Última atualização: 21/05/2025

A sua privacidade é importante para nós. Esta Política de Privacidade explica como o aplicativo Géti coleta, usa, armazena e protege seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018).

1. Dados que coletamos

Coletamos os seguintes dados fornecidos por você:

  • Nome completo
  • CPF
  • E-mail
  • Telefone
  • Data de nascimento
  • Dados de viagem (origem, destino, datas, frota)
  • Dados de empresas (CNPJ, razão social, CEP, cadastur, etc.)

Também coletamos dados automaticamente para fins estatísticos, como:

  • Modelo do dispositivo
  • Sistema operacional
  • Ações no app
  • Endereço IP (anonimizado)

2. Finalidade da coleta

Usamos os dados para:

  • Organizar viagens em grupo
  • Conectar passageiros e empresas
  • Emitir orçamentos e controlar pagamentos
  • Gerar relatórios e oferecer suporte
  • Melhorar o app e a segurança

3. Compartilhamento de dados

Seus dados não são vendidos. Compartilhamos apenas com:

  • Empresas de transporte escolhidas por você
  • Autoridades, se exigido por lei

4. Armazenamento e segurança

Os dados são armazenados no Firebase (Google Cloud) com criptografia e boas práticas de segurança.

5. Seus direitos

Você pode:

  • Acessar e corrigir seus dados
  • Revogar consentimentos
  • Solicitar exclusão da conta

Para isso, envie um e-mail para: getifretagem@gmail.com

6. Uso de terceiros

Utilizamos serviços como:

  • Firebase Authentication
  • Firebase Firestore
  • Stripe (pagamentos)

7. Alterações nesta política

Podemos atualizar esta política. Avisaremos você em caso de mudanças significativas.
